# 評価の実行方法

収集した情報を評価する手順。

## 3層分類

| Layer | 対象 | アクション |
|:-----:|------|-----------|
| 1 | 90%のノイズ | 無視 |
| 2 | 変化検知のみ | 記録・週次サマリ |
| 3 | 深掘り対象 | 30-90分で実験 |

## 手順

1. `evaluators/` のスクリプトを実行
2. 結果を確認
3. Layer 3 なら実験へ

## 判断基準

- **能力変化**: 新モデル、新機能、Agent拡張
- **制限解除**: コンテキスト長、レート制限緩和
- **価格変化**: 料金改定、新プラン
