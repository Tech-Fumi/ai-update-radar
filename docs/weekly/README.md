# Weekly Reports - 週次サマリ

毎週の検知・評価結果をまとめる。

## フォーマット

ファイル名: `YYYY-WXX.md`（例: `2025-W51.md`）

```markdown
# Week XX (YYYY-MM-DD 〜 YYYY-MM-DD)

## サマリ

| カテゴリ | 検知数 | Layer 3 | 採用 |
|---------|:------:|:-------:|:----:|
| 能力変化 | X | X | X |
| 制限解除 | X | X | X |
| 価格変化 | X | X | X |

## Layer 3（深掘り対象）

### 1. {タイトル}
- ソース:
- カテゴリ:
- スコア: X/10
- 判断: 実験 / 様子見
- 理由:

## Layer 2（検知のみ）

- {タイトル} - {一行サマリ}
- ...

## 見送り決定

- {タイトル}: {理由}
- ...

## 来週のアクション

- [ ] ...
```

## 運用フロー

```
月曜: collectors/ を実行
      ↓
火〜木: evaluators/ で評価
      ↓
金曜: experiments/ で実験
      ↓
土日: 週次サマリ作成
      ↓
月曜: exports/ に成果物を出力
```
