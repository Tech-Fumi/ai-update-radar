# v0.2.2-fix-task-embed-summary

Fix Task テンプレートに Summary Card を自動埋め込み。開いた瞬間に「何を直すか」が分かる。

## Added

### Summary Card セクション自動挿入

Fix Task 作成時、summary_card が存在すれば以下を自動で埋め込み:

```markdown
### Summary Card (AI判断)

- **Decision**: 要再試行
- **Hypothesis**: タイムアウトが原因
- **Key Points**:
  - TIMEOUT エラー
  - リトライで成功の可能性
- **Evidence**:
  - `conversation.jsonl` L120-145: TIMEOUT 発生箇所
```

| フィールド | 制限 |
|-----------|------|
| key_points | 最大 5 つ |
| evidence_links | 最大 3 つ |

### 表示条件

| 条件 | 動作 |
|------|------|
| summary_card あり | セクション挿入 |
| summary_card なし | 従来通り（セクションなし） |

## 効果

| Before | After |
|--------|-------|
| Fix Task を開いて「何これ？」 | 開いた瞬間に AI 判断が見える |
| ログを読んで状況把握 | decision + evidence で即理解 |
| 修正方針を0から考える | hypothesis がヒントになる |
