# v0.2.0-evidence-links

Summary Card の根拠リンク強化。decision/hypothesis を「検証可能」にする。

## Added

### evidence_links フィールド（構造化根拠）

```json
{
  "evidence_links": [
    {
      "artifact": "conversation.jsonl",
      "line_from": 120,
      "line_to": 145,
      "note": "TIMEOUT エラーが発生"
    }
  ]
}
```

| フィールド | 型 | 説明 |
|-----------|-----|------|
| artifact | string | 対象ファイル（conversation.jsonl, stdout.log, stderr.log, patch.diff） |
| line_from | int | 開始行番号（1-indexed） |
| line_to | int | 終了行番号 |
| note | string | 根拠の要約 |

### UI: クリック可能な根拠リンク

- evidence_links をボタンとして表示
- クリックで対応する artifact セクションを開く
- 行番号（L120-145 形式）を表示
- note を補足説明として表示

### 行番号付き会話プレビュー

- summary_card_generator が行番号付きで会話ログを出力
- AI が行番号を参照して evidence_links を生成可能に

## Changed

- `summary_card.schema.json` を v0.2.0 に更新
- evidence（旧形式）は後方互換のため残存

## 効果

| Before | After |
|--------|-------|
| decision を見て「ほんとか？」 | evidence_links をクリックして即検証 |
| ログを全部読む必要あり | 該当箇所に直接ジャンプ |
| 判断の根拠が曖昧 | 行番号で特定可能 |
