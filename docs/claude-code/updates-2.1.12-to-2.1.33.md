# Claude Code 新機能まとめ（v2.1.12 → v2.1.33）

作成日: 2026-02-06

---

## 重要度: 高

### Opus 4.6 対応（v2.1.32）
- Claude Opus 4.6 が利用可能に

### マルチエージェント（v2.1.32, v2.1.33）
- エージェントチーム機能（research preview、要 experimental フラグ）
- tmux 経由のチームメイトセッション修正
- `TeammateIdle`, `TaskCompleted` フックイベント追加

### Auto Memory（v2.1.32, v2.1.33）
- AI が自動的にメモリを記録・参照する機能
- `user`, `project`, `local` スコープ対応

### Task 管理システム（v2.1.16, v2.1.19, v2.1.20）
- 依存関係付きタスク管理（TaskCreate, TaskUpdate, TaskList, TaskGet）
- タスク削除機能
- `CLAUDE_CODE_ENABLE_TASKS` 環境変数でトグル

### npm 非推奨・native installer 移行（v2.1.15）
- npm インストールが非推奨に
- native installer への移行を案内

### `--from-pr` セッション復帰（v2.1.27）
- PR 番号からセッションを再開可能
- PR 自動リンク機能

---

## 重要度: 中

### PDF 対応強化（v2.1.30）
- `pages` パラメータで特定ページ範囲を指定可能
- 大きな PDF は軽量リファレンスを返す

### `/debug` コマンド（v2.1.30）
- セッションのトラブルシューティング用デバッグコマンド

### トークンメトリクス（v2.1.30）
- Task ツールの結果にトークン数、ツール使用回数、所要時間を表示

### キーバインドカスタマイズ（v2.1.18）
- `/keybindings` でキーボードショートカットを完全カスタマイズ可能
- コードシーケンス対応

### セッション管理改善（v2.1.31）
- 終了時にセッション再開ヒントを表示
- セッション圧縮の信頼性向上

### MCP OAuth 事前設定（v2.1.30）
- Dynamic Client Registration 非対応サーバー向けに OAuth 認証情報を事前設定

### スキル自動検出強化（v2.1.32）
- `--add-dir` で追加ディレクトリからスキルを読み込み

### カスタムコマンド引数（v2.1.19）
- `$0`, `$1` の省略構文で引数アクセス

### VSCode 統合（v2.1.16, v2.1.21）
- ネイティブプラグイン管理
- Python venv 自動アクティベート
- OAuth ユーザーがリモートセッションをブラウズ可能

### IME 対応（v2.1.21）
- 日本語 IME からの全角数字入力に対応

### PR レビューステータス（v2.1.20）
- プロンプトフッターにブランチの PR 状態を表示

---

- AVX 非対応プロセッサでのクラッシュ修正（v2.1.17）
- Bedrock/Vertex の beta ヘッダー検証エラー修正（v2.1.25）
- mTLS・プロキシ対応修正（v2.1.23）
- メモリリーク修正（v2.1.14, v2.1.16）

### UI/UX
- `spinnerVerbs` 設定でスピナーメッセージをカスタマイズ（v2.1.23）
- Bash 履歴からのオートコンプリート（v2.1.14）
- Vim ノーマルモードで矢印キー履歴（v2.1.20）
- React Compiler による UI レンダリング改善（v2.1.15）

## アップグレード推奨

**即時アップグレード推奨。** native installer により自動更新されるため、手動操作は不要。

```bash
# 現在のバージョン確認
claude --version

# 手動更新が必要な場合
claude update
```
