# 評価基準（Evaluators）

評価ロジックの仕様。

## 3層分類の基準

### Layer 1（無視）

- マイナーバグ修正
- ドキュメント更新のみ
- UI変更のみ

### Layer 2（記録）

- 新機能追加（軽微）
- パフォーマンス改善
- 新モデルバリアント

### Layer 3（実験）

- 新モデル（メジャー）
- 価格変更
- コンテキスト長変更
- 新API機能

## 評価スクリプト

```
evaluators/
├── classify.py      # Layer 分類
├── impact.py        # 影響度判定
└── notify.py        # 通知
```
